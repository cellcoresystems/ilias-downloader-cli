Wenn Sie ILIAS Downloader 3 nutzten möchten, müssen Sie die folgenden Punkte akzeptieren:

  • Dieses Programm wird Ihr Passwort nicht speichern.
  • Dieses Programm wird Ihren Loginnamen und Ihr Passwort nur an den von Ihnen angegebenen Server senden.
  • Ihr Passwort wird im Arbeitsspeicher dieses Computers nicht verschlüsselt gespeichert.
    Ein Schadprogramm könnte den Arbeitsspeicher auslesen und so an Ihre Logindaten gelangen.
  • Im nächsten Schritt müssen Sie Ihren Ilias Server angeben. Bitte achten Sie darauf, dass die Adresse mit https:// beginnt.
    Das bewirkt eine gesicherte Verbindung zwischen ILIAS Downloader 3 und Ihrem Ilias Server.
  • Die Nutzung und die Weitergabe von ILIAS Downloader 3 ist kostenlos und erwünscht.
  • Der Verkauf von ILIAS Downloader 3 ist verboten.
  • Beim Programmstart wird auf Updates überprüft. Dabei wird lediglich der Programmname an den Updateserver gesendet. Ihre Logindaten oder andere persönliche Daten werden nicht übertrage. Wenn Sie die Updatefunktion ausschalten möchten, starten Sie das Programm mit dem Parameter noupdate.
